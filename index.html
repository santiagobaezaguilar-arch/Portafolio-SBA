Diccionario CSS

Conceptos básicos de CSS:

  • PX: Unidades absolutas para definir tamaños fijos.
  • %: Unidades relativas al contenedor padre.
  • EM: Unidades relativas al tamaño de fuente del elemento.
  • REM: Unidades relativas al tamaño de fuente raíz.
  • VW: Unidades relativas al ancho de la ventana.
  • VH: Unidades relativas a la altura de la ventana.
  • AUTO: Tamaño automático basado en contenido.
  • Clases: Agrupan elementos con características similares.
  • ID: Identifican elementos únicos.
  • Elemento: Partes que componen una página web (texto, imágenes, botones, etc.).
