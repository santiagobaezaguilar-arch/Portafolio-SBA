# 🌐 Portafolio Web Personal

## 📌 Descripción

Este proyecto es un **portafolio web personal** desarrollado con HTML y CSS, cuyo objetivo es presentar información sobre desarrollo web, destacando áreas como **frontend y backend**, además de mostrar contenido estructurado de forma moderna y visualmente atractiva.

La página incorpora múltiples efectos visuales y animaciones para mejorar la experiencia del usuario, simulando un diseño profesional similar al de aplicaciones reales.

---

## 🎯 Objetivo del Proyecto

* Practicar habilidades en **HTML y CSS**
* Aplicar conceptos de **animaciones y diseño moderno**
* Crear una interfaz atractiva tipo portafolio
* Comprender la estructura de páginas web reales

---

## 🧩 Características Principales

### 🎨 Diseño

* Fondo con **degradados (linear-gradient)**
* Efecto **glassmorphism (blur con transparencia)**
* Uso de sombras (`box-shadow`) para profundidad

### 🎬 Animaciones

* `fadeIn` → aparición suave de la página
* `slideIn` → entrada desde arriba
* `slideUp` → entrada desde abajo
* `shimmer` → efecto de brillo en el header
* `glow` → brillo animado en títulos

### ✨ Interactividad

* Efectos `hover` en:

  * enlaces
  * secciones
  * imágenes
* Animaciones suaves con `transition`
* Escalado y movimiento dinámico (`transform`)

### 📦 Estructura

* Header con navegación
* Secciones informativas
* Listas y tablas estilizadas

## 🚀 Conceptos Aplicados

* Flexibilidad de diseño con `max-width`
* Posicionamiento (`relative` y `absolute`)
* Uso de pseudo-elementos (`::before`)
* Manejo de `overflow`
* Animaciones con `@keyframes`
* Transiciones suaves (`transition`)
* Filtros visuales (`filter`, `backdrop-filter`)
